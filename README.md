# 🚀 Space Predictive Analytics Center

## Projeto Acadêmico - FIAP Application Development

---

## 📋 Visão Geral

Plataforma web responsiva e moderna para monitoramento de uma missão espacial experimental. O projeto foi desenvolvido utilizando **HTML5 semântico, CSS3 com Flexbox e Media Queries, e JavaScript puro (Vanilla JS)** para fins educacionais.

---

## 🎨 Recursos Implementados

### Dashboard de Monitoramento

- 4 métricas principais em cards interativos
- Barras de progresso dinâmicas
- Cores adaptadas ao status (verde, amarelo, vermelho)
- Atualização automática a cada 3 segundos
- Botão para atualizar manualmente

### Monitoramento de Sensores

- 6 sensores diferentes com leituras específicas
- Nomes descritivos (Temperatura Externo, Pressão, Radiação, etc)
- Status visual (Ativo/Inativo)
- Barras de progresso animadas
- Leituras em tempo real

### Formulário de Relatório Técnico

- Nome completo do engenheiro
- Email com validação
- Tipo de relatório (Select)
- Nível de prioridade (Select)
- Descrição detalhada
- Recomendação de ação
- Feedback visual após envio (sucesso/erro)

### Análise Inteligente (IA)

- Status em tempo real
- Gerador de recomendações aleatórias
- Barra de confiança animada
- Simulação realista de processamento
- Interface lateral responsiva

### Tema Claro e Escuro

- Ícone de toggle no header
- Persistência em localStorage
- Transições suaves
- 20 variáveis CSS por tema
- Cores otimizadas para acessibilidade

### Navegação

- Header sticky (fixa no topo)
- Menu responsivo
- Links para seções com scroll suave
- Status de conexão em tempo real

### Data e Hora

- Atualização automática a cada segundo
- Formato localizado (pt-BR)
- Exibição no footer e dashboard
- Timestamp de última atualização

## 🚀 Como Usar

### 1. **Abrir o Projeto**

```bash
# Simplesmente abra o arquivo index.html no navegador
# Pode ser via duplo clique ou arrastar para o navegador
```

### 2. **Recursos Disponíveis**

**Dashboard**

- Visualize métricas em tempo real
- Clique em "🔄 Atualizar Dados" para refresh manual
- As métricas se atualizam automaticamente a cada 3 segundos

**Sensores**

- Deslize para a seção de sensores
- Veja o status de 6 sensores diferentes
- Verifique as leituras em tempo real

**Relatório Técnico**

- Preencha o formulário com dados técnicos
- Selecione tipo e prioridade
- Clique em "✓ Enviar" para submeter
- Mensagem de sucesso/erro aparecerá abaixo

**Análise IA**

- Clique em "🧠 Analisar com IA"
- Sistema gera recomendações aleatórias
- Exibe percentual de confiança

**Tema**

- Clique no ícone 🌙/☀️ no header
- Tema é salvo automaticamente
- Página mantém tema ao recarregar

### 3. **Responsividade**

- **Desktop**: Layout completo com sidebar
- **Tablet**: Layout em 2 colunas
- **Mobile**: Layout em coluna única otimizado
